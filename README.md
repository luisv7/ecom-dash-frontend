# eCommerce Dashboard
- Create your online store/catalog in minutes with eCom Dash. 

## User Stories
- As a user, I want to create, update and delete a product.
- As a user, I want to preview view my website/catalog

## Technologies Used
- MongoDB/Mongoose
- Express.js
- React.js
- Node.js
- CSS
- Material UI
